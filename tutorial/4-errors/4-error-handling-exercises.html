Error Handling Exercises

Try these exercises to practice error handling in KelpyShark:

1. Handle File Not Found

  • Try to read "missing.txt" and print a custom error message if it fails.

2. Throw a Custom Error

  • Write a function that throws an error if the input is negative.

3. Catch and Print Error Details

  • Throw a dictionary error and print its type and message in the catch block.

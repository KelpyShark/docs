# Method Exercises

Try these exercises to practice simulating methods in KelpyShark:

## 1. Update an Attribute
- Write a function `set_color(car, color)` that sets the `color` attribute of a car dictionary.

## 2. Get a Value
- Write a function `get_balance(account)` that returns the `balance` from an account dictionary.

## 3. Chain Functions
- Write two functions, `increment(x)` and `double(x)`, and use them together to compute `double(increment(3))`.

## Solutions
Try to solve these on your own before looking up the answers!

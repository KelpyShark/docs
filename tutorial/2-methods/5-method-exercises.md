<h1 id="method-exercises">Method Exercises</h1>
<p>Try these exercises to practice simulating methods in KelpyShark:</p>
<h2 id="1-update-an-attribute">1. Update an Attribute</h2>
<ul>
<li>Write a function <code>set_color(car, color)</code> that sets the <code>color</code> attribute of a car dictionary.</li>
</ul>
<h2 id="2-get-a-value">2. Get a Value</h2>
<ul>
<li>Write a function <code>get_balance(account)</code> that returns the <code>balance</code> from an account dictionary.</li>
</ul>
<h2 id="3-chain-functions">3. Chain Functions</h2>
<ul>
<li>Write two functions, <code>increment(x)</code> and <code>double(x)</code>, and use them together to compute <code>double(increment(3))</code>.</li>
</ul>

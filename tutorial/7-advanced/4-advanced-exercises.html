Advanced Exercises

Try these exercises to practice advanced features in KelpyShark:

1. Function as Argument

  • Write a function apply(f, x) that calls f(x).

2. Import and Use a Module

  • Import the math module and print the square root of 25.

3. Simulate a Macro

  • Write a function that takes an expression as a string and prints it before evaluating (simulate macro behavior).

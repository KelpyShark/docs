<h1 id="modules">Modules</h1>
<p>KelpyShark supports modules for organizing code. You can import modules using the <code>import</code> statement.</p>
<h2 id="example">Example</h2>
<pre><code class="lang-kelpyshark"><span class="hljs-keyword">import</span> math
<span class="hljs-built_in">print</span> math.<span class="hljs-built_in">sqrt</span>(<span class="hljs-number">16</span>)<span class="hljs-meta">   # 4</span>
</code></pre>
<h2 id="summary">Summary</h2>
<ul>
<li>Use <code>import</code> to include modules.</li>
<li>Standard library and user modules are supported.</li>
</ul>

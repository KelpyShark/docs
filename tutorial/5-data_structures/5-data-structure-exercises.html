Data Structure Exercises

Try these exercises to practice using lists and dictionaries in KelpyShark:

1. List of Numbers

  • Create a list of the numbers 1 to 5 and print the third element.

2. Dictionary Lookup

  • Create a dictionary for a book with keys "title" and "author". Print the author.

3. Nested Access

  • Create a list of dictionaries for three students. Print the name of the second student.

Solutions

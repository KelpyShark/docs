<h1 id="introduction-to-data-structures">Introduction to Data Structures</h1>
<p>Data structures are ways to organize and store data so you can use it efficiently. In KelpyShark, the most common data structures are lists and dictionaries. Understanding these will help you write programs that are clear, fast, and easy to maintain.</p>
<h2 id="lists">Lists</h2>
<ul>
<li>Ordered collections of values (like arrays in other languages).</li>
<li>Useful for storing sequences, such as numbers, names, or objects.</li>
</ul>
<h2 id="dictionaries">Dictionaries</h2>
<ul>
<li>Collections of key-value pairs for mapping data.</li>
<li>Useful for looking up values by a unique key, like a name or ID.</li>
</ul>
<h2 id="why-use-data-structures-">Why Use Data Structures?</h2>
<ul>
<li>To group related data together.</li>
<li>To make your code more readable and maintainable.</li>
<li>To perform operations like searching, sorting, and filtering efficiently.</li>
</ul>

<pre><code class="lang-kelpyshark"><span class="hljs-attr">nums</span> = [<span class="hljs-number">1</span>, <span class="hljs-number">2</span>, <span class="hljs-number">3</span>, <span class="hljs-number">4</span>, <span class="hljs-number">5</span>]

<h2 id="summary">Summary</h2>
<ul>
<li>Use lists and dictionaries for most data storage needs.</li>
<li>Mastering these structures is essential for effective programming in KelpyShark.</li>
</ul>

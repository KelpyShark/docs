# Introduction to Data Structures

Data structures are ways to organize and store data so you can use it efficiently. In KelpyShark, the most common data structures are lists and dictionaries. Understanding these will help you write programs that are clear, fast, and easy to maintain.

## Lists
- Ordered collections of values (like arrays in other languages).
- Useful for storing sequences, such as numbers, names, or objects.

## Dictionaries
- Collections of key-value pairs for mapping data.
- Useful for looking up values by a unique key, like a name or ID.

## Why Use Data Structures?
- To group related data together.
- To make your code more readable and maintainable.
- To perform operations like searching, sorting, and filtering efficiently.

## Examples
```kelpyshark
nums = [1, 2, 3, 4, 5]
names = ["Alice", "Bob", "Charlie"]
info = {"name": "Alice", "age": 30, "city": "Paris"}
scores = {"math": 95, "science": 88}
```

You can also combine lists and dictionaries for more complex data:
```kelpyshark
students = [
	{"name": "Alice", "age": 30},
	{"name": "Bob", "age": 22}
]
```

## Summary
- Use lists and dictionaries for most data storage needs.
- Mastering these structures is essential for effective programming in KelpyShark.

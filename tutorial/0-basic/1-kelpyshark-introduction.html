Introduction to KelpyShark

KelpyShark is a general-purpose programming language designed for readability, versatility, and ease of use. It is suitable for:

  • Scripting and automation
  • Application development
  • Learning programming concepts
  • Building libraries and tools

Philosophy

  • Beginner-friendly: Simple, readable syntax
  • Powerful: Supports advanced features for experts
  • Flexible: Dynamic typing, first-class functions, and collections
  • Modern: Cross-platform, compiles to multiple targets

Key Features

  • Simple, readable syntax (inspired by Python, C, and JSON)
  • Dynamic typing
  • Curly braces for code blocks
  • First-class functions
  • Built-in lists and dictionaries
  • String interpolation
  • Cross-platform (Windows, Linux, macOS)
  • Transpiles to C, JavaScript, Java, and C#
  • Native binary compilation
  • Standard library and package manager

# Welcome to KelpyShark

KelpyShark is a modern, beginner-friendly programming language designed to be readable, powerful, and fun. This tutorial will guide you from your very first program to advanced features, with clear explanations and practical examples.

## Who is this for?

- **Absolute beginners**: No prior programming experience needed. Start with the basics and follow the examples step by step.
- **Experienced programmers**: Learn a new, expressive language with modern features and familiar syntax.
- **Experts**: Explore advanced features, cross-language compilation, extensibility, and performance.

## What will you learn?

- The basics: variables, data types, operators, control flow, functions, and collections
- Advanced topics: modules, error handling, file I/O, classes, methods, and more
- How to use the standard library and package manager
- How to build, run, and distribute KelpyShark programs

## How to use this tutorial

- Work through each section in order, or jump to topics of interest
- Try the code examples in your own files or in the interactive REPL (`kelpy repl`)
- Use the [Quick Reference](../9-reference/0-navigation.md) for fast lookups

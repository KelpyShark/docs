# IO Streams Exercises

Try these exercises to practice IO operations in KelpyShark:

## 1. User Greeting

- Ask the user for their name and print a greeting.

## 2. Echo Input

- Read a line from the user and print it back.

## 3. File Copy

- Read the contents of "input.txt" and write them to "output.txt".

## Solutions

Try to solve these on your own before looking up the answers!
